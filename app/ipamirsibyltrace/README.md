An example IPAMIR application for SibylSat intermediate MaxSAT traces.

# Compilation

The application can be compiled using the provided Makefile as follows:

```
make
```

# Running the Application

The application can be run on a trace file as follows:

```
./ipamirsibyltrace <trace_file.wcnf>
```

# Inputs

The inputs can be downloaded [here](https://www.cs.helsinki.fi/group/coreo/MSE2026/sibyltrace-inputs.zip).
